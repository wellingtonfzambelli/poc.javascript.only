# About
This project demonstrates the use of pure javascript  ECMAScript 6+ and Vite Compiler

# Stacks of this project
- Javascript
- ECMAScript 6+
- Vite - build tool
- Visual Studio Code
- Google Chrome

# Setting up the vite
1 - Open the terminal -> go to your project directory 
2 - Type the npm command

```terminal
npm create vite@latest
```
